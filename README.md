# MazeGameSword

### 一、 游戏剧情（简单概要）：
勇者为了打败魔王需要收集三个碎片，合成圣剑，才能打败魔王。

### 二、 游戏设计：
#### 总场景
1.	开篇的剧情
2.	3个普通关
3.	BOSS战关卡
4.	结局的剧情
5.	主菜单
6.	死亡场景

### 三、 细节设计
#### 1.	开篇的剧情
   
#### 2.	3个普通关
   
（1）	角色

移动的控制

（2）	敌方

实现追踪功能

（3）	场景

（使用坐标来定位，贴图为每一个砖墙或地板）

提前预输入地图，实现碰撞

（4）	一些物体

药水+1生命值；金币+1得分；钥匙（改变状态hasKey）；出口，有钥匙则进入下一个场景；（传送门考虑）

圣剑碎片（bool数组，当三个都获取时合成圣剑）圣剑可以对BOSS造成伤害（按j发射弹幕，追踪Boss）

（1）	Boss不近身攻击，通过弹幕的生成，移动攻击。

（2）	地图在下半，封闭，Boss从上方攻击，只有弹幕可跨越两个区域
